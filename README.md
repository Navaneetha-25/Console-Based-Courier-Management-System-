## Courier Management System (CMS)

A simple Java-based Courier Management System that simulates how courier companies manage parcels, agents, and tracking updates.
The system supports booking parcels, assigning delivery agents, tracking shipment progress, and maintaining parcel history.

## Features

 Register and manage courier parcels
 Manage courier agents
 Book new parcels with initial status
 Assign parcels to valid courier agents
 Update parcel status during shipment
 Add tracking updates with notes
 View full parcel tracking history
 Cancel parcels and remove related updates
 Display parcels by agent
 Display all parcels with current status

## Purpose of the Project

This project was built to:

Practice object-oriented programming (OOP) concepts
Implement service-layer business logic
Use custom exceptions for validation
Simulate a real-world courier workflow
